Hexo框架下用Next主题美化

发表于 2019-09-03 更新于 2019-09-06

Next主题的安装

Hexo安装主题，只需要将主题文件拷贝至博客所在目录的themes目录下，修改相关配置文件即可生效。博客所在目录下打开git bash，再通过Git clone https://github.com/theme-next/hexo-theme-next themes/next即可完成。

启用主题

打开根目录下的_config.yml,查找theme字段，将字段改为theme: next(冒号:之后要有空格分隔，否则无效) ，之后通过hexo g和hexo s，再在浏览器中访问localhost:4000即可本地预览主题效果。（注意4000端口号是否被占用，更改端口号：$ hexo s -p 5000）

设置语言

博客框架默认的语言是英文，前往/themes/next/languages，查看当前NexT版本简体中文对照文件的名称是zh-Hans还是zh-CN。

再前往根目录下的_config.yml，查找language，设置成language: zh-Hans或language: zh-CN，即显示简体中文。
